Program #4: Develop a program to print your name in two lines
		I/P:			O/P:
		HariKrishna		Hari
					Krishna
		//NameInTwoLines.java			
		class NameInTwoLines {
			public static void main(String[] args) {
				System.out.println("Hari\nKrishna");
			}			//\n is called escape sequence character
		}				//it is used for inserting one new line
						//it moves its right side characters to
						//the next line
		cmd>javac NameInTwoLines.java
			|-> NameInTwoLines.class
			
		cmd>java NameInTwoLines
			|-> Hari
			    Krishna
			
Program #5: Develop a program to print your name as below
		I/P	O/P
		===	====				In this program we must use
		 H	H				the esc character \t for inserting
		 A		A			one tab space (8 spaces)
		 R			R
		 I				I
		
		//NamePattern.java
		class NamePattern {
			public static void main(String[] args) {
				System.out.println("H");
				System.out.println("\tA");
				System.out.println("\t\tR");
				System.out.println("\t\t\tI");
			}
		}
		
		cmd>javac NamePattern.java 
			|-> NamePattern.class
			
		cmd>java NamePattern
			|-> H
			            A
			                    R
			                            I

Program #7: Develop a program to print your name in single quote and double quote
		I/P		O/P
		====		===			For printing ' and "	
		Hari Krishna	'Hari Krishna'		we must use esc characters \' & \"
		Hari Krishna	"Hari Krishna"
		
		//NameInQuotes.java
		class NameInQuotes {
			public static void main(String[] args) {
			    System.out.println("\'Hari Krishna\'");
			    System.out.println("\"Hari Krishna\"");
			}
		}
